# 🌌 AsteriaMind


AsteriaMind is an experimental cognitive architecture.

The goal is not only to make machines answer questions,
but to explore how a system can:

- learn from experience
- discover uncertainty
- generate hypotheses
- verify itself
- evolve its own capabilities
